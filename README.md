## 📌 Project Overview

**E-CarStart** is an interactive web application that visualizes and analyzes **Electric Vehicle (EV)** data using **Python Flask** and **Tableau Public**.

The application provides meaningful insights into India's EV ecosystem through interactive dashboards and stories, enabling users to explore registration trends, manufacturer performance, vehicle categories, and EV adoption.

---

## ✨ Features

- 🏠 Responsive Home Page
- 📊 Interactive Tableau Dashboard
- 📖 Interactive Tableau Story
- 🌐 Flask-based Web Application
- 📱 Mobile Responsive Design
- 📈 EV Registration Analysis
- 🚗 Vehicle Category Insights
- 🏭 Manufacturer-wise Analysis
- 📍 State-wise EV Trends
- ⚡ Clean and Modern User Interface

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/nharshitha19/TableauFlaskProject
```

### Navigate to the Project

```bash
cd Electric-Vehicle-Analytics
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Flask Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
