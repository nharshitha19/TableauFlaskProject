About This Project

Electric Vehicle Analytics is a web application developed using Python, Flask, Bootstrap, and Tableau Public.

The project provides interactive dashboards and stories to analyze electric vehicle adoption, state-wise registrations, charging infrastructure, and market trends across India.

It demonstrates how business intelligence and data visualization can be used to transform raw data into meaningful insights for decision-making.
