About the Project

This project analyzes the growth and adoption of Electric Vehicles (EVs) in India using interactive Tableau visualizations. The website is built with Flask and Bootstrap, while Tableau Public is used to present dashboards and stories.

Project Objectives

  • Analyze EV adoption trends in India.
  • Visualize state-wise EV registrations.
  • Understand market growth through interactive dashboards.
  • Present insights using Tableau Stories.
